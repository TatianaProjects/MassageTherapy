# Renew & Relax

A modern React SPA for a massage and wellness business.

## Live Demo

[https://your-netlify-link.netlify.app](https://renewandrelaxmassage.netlify.app/)

## Features

- Smooth Navigation – Seamless page transitions powered by React Router.
- Fully Responsive – Optimized UI that looks great on all devices (mobile, tablet, desktop).
- Smart Service Filtering – Quick and dynamic filtering to find services instantly.
- Dynamic Content Rendering – Content updates automatically and efficiently.
- Interactive Reviews Slider – Modern carousels for client feedback and testimonials.
- Interactive Accordions – Smooth toggle elements for FAQs.
- Contact & Location – Integrated Google Maps for easy location tracking.


## Technologies

- React
- React Router
- JavaScript
- CSS3
